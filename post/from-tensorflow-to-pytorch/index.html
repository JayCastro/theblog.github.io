I have been using TensorFlow since 2016, but I have started using PyTorch more and more in my recent projects. Although the key concepts of both frameworks are pretty similar, especially since TF 2.0, I wanted to make sure that I don't overlook some critical difference in PyTorch. Therefore, I read through all of the current 23 (TODO) official PyTorch tutorials and the 13 notes in the PyTorch documentation.

1. % can output negative numbers

The % operator calculates the remainder of the division, which does not necessarily have to be in [0, n-1]. 
int positive = 23 % 7; // will be 2
int negative = -17 % 7; // will be -3
